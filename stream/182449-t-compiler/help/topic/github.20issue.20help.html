What exactly do you need? If you have specific questions you'll more likely to get answers. Just one tip (I saw your question in the issue): use ./x.py build --stage 1 library/std to build, then do rustup toolchain link stage1 build/x86_64-unknown-linux-gnu/stage1. After that keep rebuilding with the same command (no need for the rustup command again) and use the compiler you built with rustc +stage1 or cargo +stage1.
