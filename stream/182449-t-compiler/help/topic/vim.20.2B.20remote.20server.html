simulacrum (Jul 15 2021 at 22:40):

I have this for setting up the remote end of repositories:

#!/bin/bash

set -Eeuxo pipefail

CUR_DIR=$(basename `pwd`)
DIR=${1:-$CUR_DIR}

ssh $REMOTE_SERVER git init Build/$DIR
ssh $REMOTE_SERVER git -C Build/$DIR commit --allow-empty -m "t"

cp ~/Edit/build-script.sh ./build.sh
chmod u+x ./build.sh

and build-script.sh is like this -- but it usually ends up with some minor per-project edits around setting or not setting incremental etc. 

#!/bin/bash

set -e

HOST=$REMOTE_SERVER
CMD="cargo check"
DIR="Build/$( cd "$( dirname "${BASH_SOURCE[0]}" )" && basename `pwd` )"
git push -q -q -f $HOST:$DIR $((git stash create --quiet; echo HEAD) | head -n 1):master
CMD="cd $DIR && git checkout -qq --force --detach master && (CARGO_INCREMENTAL=1 ${@:-$CMD})"
if test "$COLOR" == "vim"; then
    ssh -q $HOST "$CMD"
else
    ssh -q -tt $HOST "$CMD"
fi
scp -q $HOST:$DIR/Cargo.lock .
