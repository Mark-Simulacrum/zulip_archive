Topic: find certain type's drop function MIR

Shihao Xia (Oct 31 2021 at 19:38):

Hi there, suppose we have struct ABC in other crate D.

struct ABC{...}

impl Drop for ABC {
    func drop(&mut self) {...}
}

If we have a new cargo project, in its main.rs

...
let abc = ABC{...}

in MIR:

let _1: ABC
drop(_1)

When I traverse the MIR of main.rs. any method we can find the drop implementation of struct ABC at crate D?

tm (Oct 31 2021 at 20:23):

For the drop glue MIR, i.e., the one that calls <ABC as Drop>::drop method and then drops individual fields, that would be something like Instance::resolve_drop_in_place(tcx, ty) and then tcx.instance_mir(instance).
