You have to include the host platform in the target list. Otherwise build scripts and proc macros can't be built as they run on the host.

bjorn3 (Mar 28 2022 at 13:30):

By the way have you seen cargo build -Zbuild-std before? It allows you to build the standard library for any target without building rustc yourself.
