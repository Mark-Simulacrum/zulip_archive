If I have a DefId, whats a reliable way to check if the item is defined in std, core or alloc?

bjorn3 (May 02 2021 at 11:11):

There is no "reliable" way as none of the crates are hard coded into the compiler. You can estimate it by checking tcx.crate_name(def_id.krate), but it is possible for a user crate to be named the same as any of these.
