mark-i-m (Jun 08 2020 at 23:50):

Some other possible way to get started:

  • Working on one of the many other related projects (e.g. chalk if you're interested in traits). A list of working groups: https://rust-lang.github.io/compiler-team/working-groups/
  • You could join the ICE-breakers group, which requires very little context but is extremely useful: https://rustc-dev-guide.rust-lang.org/ice-breaker/cleanup-crew.html
