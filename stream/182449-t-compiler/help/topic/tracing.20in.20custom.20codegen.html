Matthew Jasper (Oct 12 2021 at 12:50):

So for debug and trace logging to work you need to set

[rust]
debug = true

in config.toml. RUSTC_LOG was also broken for a few days recently, so check that you have #89743.

Matthew Jasper (Oct 12 2021 at 12:52):

debug = trueenables debug assertions as well, if you only want tracing then you can set debug-logging = true instead
