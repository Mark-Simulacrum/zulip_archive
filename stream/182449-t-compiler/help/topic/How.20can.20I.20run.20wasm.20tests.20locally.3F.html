Eric Huss (Sep 01 2021 at 15:48):

I would normally use the docker environment (use --dev for an interactive terminal). However, if you don't have docker or if that takes too long, the minimum necessary to do it in your local environment is:

  • Install node.js
  • Add wasm32-unknown-unknown to build.target (or specify --target=wasm32-unknown-unknown).
  • Set lld = true

Also keep in mind that only a subset of tests work on wasm.
