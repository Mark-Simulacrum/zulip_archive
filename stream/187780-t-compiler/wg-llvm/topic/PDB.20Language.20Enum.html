Topic: PDB Language Enum

Arlo Siemsen (Dec 08 2021 at 19:18):

Rust currently shows up as language "MASM" in the PDB format because no enumerator value was available. I've been working to get a value assigned to Rust by Microsoft, and I just submitted a review to LLVM to use the new value.

Here's the LLVM review that makes this change.
https://reviews.llvm.org/D115300

I believe no additional changes will be required on the Rust side (other than updating LLVM). Please let me know if I should create an issue, or do something else on the Rust side to help track this.
