Alex Parrill (Apr 11 2022 at 17:47):

Is this safe, or is it violating some type aliasing rules?

fn fix_slice<T, const D: usize>(slice: &[T]) -> Option<&[T: D]> {
    if slice.len() == D {
        Some(unsafe { &*(slice.as_ptr() as *const [T;D]) })
    } else { None }
}

scottmcm (Apr 12 2022 at 01:53):

See also https://doc.rust-lang.org/nightly/std/primitive.slice.html#method.as_chunks_mut, which is a more general form of this transform:

pub fn fix_slice<T, const D: usize>(slice: &mut [T]) -> Option<&mut [T; D]> {
    if let ([a], []) = slice.as_chunks_mut() {
        Some(a)
    } else {
        None
    }
}
