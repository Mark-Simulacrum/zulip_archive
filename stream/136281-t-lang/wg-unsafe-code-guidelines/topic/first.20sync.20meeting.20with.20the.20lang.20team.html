# Unsafe code guidelines working group sync

The first lang-team working group sync will be taking place this coming Thursday, March 14. The meeting concerns the Unsafe Code Guidelines working group. We will be presenting this write-up of the current status and plans. The goal of the meeting is both to inform the lang-team and broader community about what we've been up to, and to gather feedback on the overall direction -- not so much to debate technical minutia, though that too could happen. =)

This is an open meeting. It will be held over Zoom and will be recorded and posted to YouTube after the fact.

I hope that you all can make it!
