Lokathor (Sep 28 2020 at 16:29):

I think one thing that has indeed stalled the progress is that there's enough the WG has nailed down that T-Lang needs to start voting on things and saying at least "yes, we approve of these things, use them as a basis for further developments". Which isn't to say that the decisions have to be set in stone, but that we have to just have some sort of confirmation/feedback.

I'd make a comparison to the code contribution process: there's a big set of PRs piling up, and some of them need to at least make it to Nightly status so that they can be used for further development. That doesn't make them instantly Stable, so we can adjust later if some sort of problem comes up.

And I mean this would be before the RFC writing process, so that people would know where to focus a first RFC's efforts.

Lokathor (Sep 28 2020 at 16:30):
