nikomatsakis (Oct 28 2021 at 13:56):

Hey @T-compiler! @Doc Jones and I had this idea for a rustc code reading club. I'm hoping that many of you would like to join for the first meeting! We had made a little website here.

The goal here is both to offer new contributors a friendly way to learn more about rustc but also to help those of us who already know rustc to learn more of it (I also think it's nice for people to see that even experienced compiler hackers get confused sometimes reading the code). To that end, I picked a bit of code to read for the first few meetings that I personally would like to know better -- the name resolution code (cc @Vadim Petrochenkov!). Would love to get suggestions for future meetings.

It occurs to me though, how would people feel about making this an "official" compiler team activity? We could make a little subteam (@Doc Jones and I would be happy to lead for now) and move the repo into rust-lang; we'd also create a t-compiler/code-reading-club stream. Both of us would definitely prefer it that way! I'll bring it up in the triage meeting today.
