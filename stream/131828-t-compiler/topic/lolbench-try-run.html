@nikomatsakis there isn’t yet a feature on lolbench to run a “try job” (also the github pages site is down bc of too much data in the repo >_<) but the whole suite can be run locally. there are some recommended settings in the repo read me and in the ansible scripts for reducing noise. what the remote runners execute is pretty straightforward and in those files. on a pretty fast machine the whole suite takes 6+ hours right now for a single toolchain, so prepare for some waiting. strongly recommend bare metal over VMs, the folks at packet have agreed to donate machines to lolbench but i haven’t set them up yet
