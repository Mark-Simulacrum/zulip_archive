jsha (Dec 15 2021 at 02:25):

A little clarification:

There's a category of tokens generally called "security keys" that just do WebAuthn / U2F. Those are mainly used for web logins. For instance, these are a great way to secure your GitHub account.

As of OpenSSH 8.2 (https://www.openssh.com/txt/release-8.2), a security key _can_ be used for SSH authentication using U2F challenges. Both client and server have be >8.2, and this isn't general purpose key storage, so I believe it can't be used for signing git commits.

The Yubico Security Key Series (https://www.yubico.com/products/) and the Google Titan are both in this category.

Yubico also has the more expensive "YubiKey 5 Series." Those have additional applets on them for GPG and PIV. The PIV applet allows you to generate and store a regular RSA or ECDSA key on the device; you can use it to store a "normal" SSH key on device. If you want to do commit signing I believe you need the 5 Series. I'm not sure from the OP whether the Yubikeys being offered are 5 Series or not. I'm guessing they're Security Keys. Either way, it's a great offer, thank you @John Naulty.

Also I'd definitely encourage anyone who wants one to just go for it; if we exhaust the supply from this offer I'm very happy to help source more. Security keys are awesome!
