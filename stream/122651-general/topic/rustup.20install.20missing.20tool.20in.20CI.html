Topic: rustup install missing tool in CI

Noah Lev (Nov 30 2020 at 03:48):

How can I install a missing tool in rustup without manually searching through nightlies? I use nightly rustfmt for a project and I want to check it in CI, but rustup toolchain install nightly && rustup +nightly component add rustfmt doesn't work when rustfmt missed a nightly because of broken toolstate. How can I solve this?

Noah Lev (Nov 30 2020 at 04:18):

This fixed it:

rustup toolchain install nightly --component rustfmt

Hope that's useful to someone else!

Noah Lev (Nov 30 2020 at 04:18):
