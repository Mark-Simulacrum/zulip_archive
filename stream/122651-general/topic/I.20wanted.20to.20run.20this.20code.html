but if I have multiple programs, how can I choose which to compile. In my src I have main.rs and main1.rs but by default it always runs main.rs. How can I run main1.rs?

bjorn3 (Oct 05 2021 at 16:51):

If you want more than one program in the same package, you have to move the main.rs for each program to the bin subdirectory of src and name it like the actual program. For example src/bin/foo.rs and src/bin/bar.rs for the programs foo and bar. You can then use cargo run --bin foo and cargo run --bin bar.
