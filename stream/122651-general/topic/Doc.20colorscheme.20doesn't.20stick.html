Eric Huss (Apr 02 2022 at 05:17):

Most browsers do not allow pages from different file locations from accessing the localstorage information needed to store settings. Some browsers allow you to turn off that restriction in the settings, though I don't particularly recommend that. You can run a server like python3 -m http.server to browse them via localhost.
