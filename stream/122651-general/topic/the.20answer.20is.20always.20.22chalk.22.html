I have a supertrait with associated types. I have a subtrait that wants to add bounds to those associated types. The bounds are not honored. 

trait Parent {
    type Name;
    fn name(&self) -> &Self::Name;
}

trait Child
where
    Self: Parent,
    Self::Name: std::fmt::Display,
{
}

fn demo(c: impl Child) {
    let name = c.name();
    println!("{name}");
}

There are workarounds, such as

trait Child: Parent<Name = Self::Name2> {
    type Name2: std::fmt::Display;
}

or

#![feature(associated_type_bounds)]
trait Child: Parent<Name: std::fmt::Display> {}
