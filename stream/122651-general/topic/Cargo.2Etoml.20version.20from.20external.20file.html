Topic: Cargo.toml version from external file

Maximilian Hristache (Nov 17 2020 at 19:30):

Is there a way to tell cargo to read the app version from an external file instead of Cargo.toml? Or some magic syntax in Cargo.toml to tell it to use the external file to read the version?

Josh Triplett (Nov 17 2020 at 19:48):

@Maximilian Hristache No, there's no current way to do that. There's a way to get Cargo to tell you the version that's in Cargo.toml, such as to compile it into your program. But if you're looking to have Cargo read the version number from somewhere else, we don't have a way to do that. (Also, there's a #t-cargo channel that may help.)
