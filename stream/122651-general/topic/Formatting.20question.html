Noah Lev (Sep 16 2020 at 23:09):

format! is Rust's equivalent of template strings in JS:

Rust

let name = "person";
let greeting = format!("Hello, {}!", name);

JavaScript

const name = "person";
const greeting = `Hello, ${name}!`;

Not sure if that's helpful, but figured I'd put an example.
