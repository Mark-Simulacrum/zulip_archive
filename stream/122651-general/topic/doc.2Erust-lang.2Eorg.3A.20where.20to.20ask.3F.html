Giacomo Stevanato (Feb 26 2021 at 10:08):

They're generated by rustdoc.

  • If you're interested to build the docs for your crate, you can use cargo doc. See the rustdoc book for more informations.
  • If you're interested to build the stdlib docs, you can use ./x.py doc --stage 1 library/std. See this page on the rustc dev book for more informations
  • If you're interested in the rustdoc internals, you can read this page on the rustdoc internal.
