Eric Huss (Aug 25 2020 at 20:30):

@Jake Goulding I have FF 80 on macOS, and the colors look fine to me. I also tried using an empty profile, and it also seemed fine. Do you happen to have the Source Code Pro font installed locally? There is a bug in FireFox that newer versions of that font don't support colors properly on macOS. More details at https://bugzilla.mozilla.org/show_bug.cgi?id=1520157. I would try turning off the "Source Code Pro" css rule in the inspector and see if that helps.
