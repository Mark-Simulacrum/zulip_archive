Conrad Ludgate (Mar 05 2022 at 21:25):

Actix/axum are fairly simple to set up. In general though, I'd recommend using normal distributed tracing techniques for inspection (wouldn't give you the ability to pause things though) - see the tracing crate for that, paired with jaeger.

If not a web interface, maybe a graphql one? Gives easy access to any data/commands you want (if you set it up), although it'll take some time to get familiar.

But I agree, you'll get better help on the community/tokio discords
