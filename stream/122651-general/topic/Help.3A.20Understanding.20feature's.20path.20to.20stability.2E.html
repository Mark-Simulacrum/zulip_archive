Joshua Nelson (Nov 22 2021 at 01:50):

@Patrick Poitras unfortunately "stable" has two meanings - 1. is the stable release channel, like 1.56 or 1.57, and 2. is "this feature can be used without a feature gate". So zip is only available on the stable release channel if you use a feature gate (and feature gates are a hard error unless you use nightly).

Joshua Nelson (Nov 22 2021 at 01:50):

See also https://rustc-dev-guide.rust-lang.org/stability.html#stabilizing-a-library-feature
