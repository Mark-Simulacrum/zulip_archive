Topic: rustup: install component for every toolchain

Is there any way to install a component with Rustup for every toolchain installed?

Joshua Nelson (Aug 07 2020 at 21:11):

rustup toolchain list | cut -d ' ' -f 1 | xargs -n1 -I {} rustup toolchain add {} --component rustfmt
