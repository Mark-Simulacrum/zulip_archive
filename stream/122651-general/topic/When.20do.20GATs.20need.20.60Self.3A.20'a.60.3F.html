Topic: When do GATs need `Self: 'a`?

Jake Goulding (Feb 27 2022 at 17:39):

I've this GAT:

pub trait Exchange<I> {
    type Output<'a>
    where
        Self: 'a;

    fn exchange(&self, idx: I) -> Self::Output<'_>;
}

I know that the Self: 'a is somewhat a temporary thing, but I don't need to place it every time. These are fine:

impl Exchange<usize> for CoreParser {
    type Output<'a> = &'a str;

    fn exchange(&self, idx: usize) -> Self::Output<'_> {
        &self.as_str()[..idx]
    }
}

impl<R> Exchange<Streaming<usize>> for Parser<R> {
    type Output<'a>
    where
        R: 'a,
    = Streaming<&'a str>;

    fn exchange(&self, idx: Streaming<usize>) -> Self::Output<'_> {
        self.parser.exchange(idx)
    }
}
