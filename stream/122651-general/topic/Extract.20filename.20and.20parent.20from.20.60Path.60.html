Actually, I ended up modifying @Josh Triplett 's suggestion:

// Trim a path down to its parent and filename if possible (e.g.
// `/aa/bb/cc/dd.rs` becomes `cc/dd.rs`), otherwise return `path`
// unchanged.
fn trim_path(path: &Path) -> &Path {
    (|| Some(path.strip_prefix(path.ancestors().nth(2)?).ok()?))().unwrap_or(path)
}

nnethercote (Dec 08 2021 at 03:46):

Nice suggestion. I now have this:

fn trim_path(path: &Path) -> &Path {
    const N: usize = 3;
    let len = path.components().count();
    if len > N {
        let mut c = path.components();
        c.nth(len - (N + 1));
        c.as_path()
    } else {
        path
    }
}
