measureme gets some love from the wider Rust community (and also a potential bug report):

https://wiki.alopex.li/WhereRustcSpendsItsTime

It appears that -Z codegen-passes is basically now made obsolute by a more precise (and hopefully more accurate) self-benchmarking method, a -Z self-profile flag with a set of tools appropriately called measureme. This outputs some binary profiling data that then gets crunched and displayed by one of measureme’s tools, summarize. Or even better, it can output ♥flame graphs♥.
