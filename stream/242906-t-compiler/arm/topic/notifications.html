davidtwco (Jun 26 2020 at 09:09):

If you were pinged on GitHub (e.g. labelling O-ARM pings some @O-ARM group in a GitHub comment) then you'd be subscribed to all further updates to the thread - which could be a little spammy if it isn't an issue that you're interested in; in contrast, pinging on Zulip (e.g. labelling O-ARM makes a new topic here and pings you in it) then you'd get a ping once but wouldn't be "subscribed" to it (e.g. wouldn't receive further notifications).
