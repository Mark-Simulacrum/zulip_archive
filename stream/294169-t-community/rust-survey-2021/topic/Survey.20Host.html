Jasun Carr (Aug 31 2021 at 18:56):

I'm still looking into what options might work in China (it's looking that most won't without a VPN to get around the firewall), but have the basic data below. Right now, based on platform features and usability, I would recommend either SurveyMonkey or SurveyHero. Qualtrics is the most robust of the options, but for a single use like this survey it would be much too cost prohibitive.
I've created free accounts to better appraise the interfaces and options within SurveyHero & SurveyMonkey, and will update regarding those ASAP.

image.png

RUST Survey Platform Assessments:
• SurveyHero (surveyhero.com)
o One survey, multiple languages
o Single output format – reduced data wrangling
o Automatic translate powered by Google
o Ability to review translations and make edits
o Manual editing capabilities (Original text visible during translation)
o Native management of RTL text
o Requires Business ($55/mo. Individual; $39/user/mo. Team [2 minimum])/Enterprise ($89/mo. Individual; $69/user/mo. Team) level subscription
 Both for the translations and for the logic required for the Rust survey
• SurveyMonkey (surveymonkey.com)
o One survey, multiple languages
o Single output format – reduced data wrangling
o All translation done outside of survey platform
 Create survey in base language (if need RTL language, it must be the default for the survey)
 Uses set of csv files that are exported (after initial survey creation) and then imported with translation – And is a beta feature.
o No automatic translate
o Requires Team Premier ($75/user/mo. w/ 3 user minimum & billed annually)
 Both for translations & logic.
 15,000 response max
• Qualtrics
o One survey, multiple languages
o Single output format – reduced data wrangling
o Automatic translate powered by Google
o Ability to review translations and make edits
o Manual editing capabilities (Original text visible during translation)
o Can also support translation uploads
o Paid translation also available (quote basis)
o Automatic language detection based on browser setting
 Can switch language at any point in the survey (additional interface option that may be distracting?)
o Allows filter based on language
o $$$$$ (Exact amount unclear – potentially starts at $1500/year?. Would require quote…)
• Google Forms
o Separate survey for each language
o Multiple survey output – requires more data wrangling
o Ease of shared editing
o Unlimited responses
o Free
• Microsoft Forms
o One survey, multiple languages
o Single output format – reduced data wrangling
o Limited to 11 languages (or 23? Conflicting information on Microsoft resources)
o Manual editing capabilities (Original text visible during translation)
o Translation done outside of survey platform
 Create survey in base language (if need RTL language, it must be the default for the survey)
 Uses set of Excel files that are exported (after initial survey creation) and then imported with translation
o Language can be user selected or set by appending language code to the URL
