I've been wondering if there is a way to just run one test from the ui test suite. Right now I'm doing cargo test --test compile-test but this runs all of the ui test suite. Adding --test-args name_of_the_test.rs like suggested by compiletest gives an erorr, just as if you add a -- before it.

flip1995 (Feb 15 2022 at 12:21):

You can do this with TESTNAME=<testname> cargo uitest. The uitest subcommand is a shortcut for --test compile-test.
