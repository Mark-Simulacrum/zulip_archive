Going through old issues... I think it would be nice to have labels for global-analysis and range-analysis. There are a lot of suggested lints that fall in these categories. I don't know if "global analysis" is the right term - I basically mean considering how a function is implemented together with how it is used. Range analysis is like " a > 1 && a < 0 doesn't make sense".
