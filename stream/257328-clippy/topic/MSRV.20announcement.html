Topic: MSRV announcement

flip1995 (Nov 25 2020 at 12:06):

With https://github.com/rust-lang/rust-clippy/pull/6201 Clippy now has a configuration option for an MSRV. You can either set it in the Clippy configuration file or on nightly (requires custom_inner_attributes feature) with #![clippy::msrv = "1.40.0"]. When this conf is set, lints that require a specific Rust version won't trigger if the MSRV isn't met.

flip1995 (Nov 25 2020 at 12:11):

These are the lints, this config option currently applies to: https://rust-lang.github.io/rust-clippy/master/index.html#msrv (more to come)
