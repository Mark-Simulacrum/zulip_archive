Eric Huss (Nov 11 2020 at 22:13):

FYI, this is roughly the commands I used to try the LLVM coverage:

cargo install rustfilt
RUSTFLAGS="-Zinstrument-coverage" LLVM_PROFILE_FILE="${PWD}/cargo%m.profraw" cargo test
llvm-profdata merge -sparse cargo*.profraw -o cargo.profdata
llvm-cov show -Xdemangler=rustfilt target/debug/cargo -instr-profile=cargo.profdata --show-line-counts-or-regions --output-dir=cov --format=html
open cov/index.html

I'm not certain if that's all correct, but it seemed to work.
