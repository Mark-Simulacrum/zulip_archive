Yechan Bae (Aug 20 2021 at 21:38):

We're happy to share that the unsafe Rust memory safety bug detection work that we've been working on, named Rudra, is finally public. We identified three bug patterns that are prevalent in unsafe Rust, devised two static analysis algorithms for the detection, and ran an ecosystem-scale analysis. Rudra identified 264 new memory safety bugs which got 78 CVE IDs - including 2 CVE-assigned bugs in the Rust standard library.

The implementation, pre-print version of the paper, and the list of bugs are now public:
https://github.com/sslab-gatech/Rudra
https://github.com/sslab-gatech/Rudra-PoC
