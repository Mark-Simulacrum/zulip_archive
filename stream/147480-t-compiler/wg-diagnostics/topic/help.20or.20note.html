Eric Huss (Sep 05 2021 at 16:54):

In a diagnostic message, when explaining to the user why this message was emitted, does that text go into a note or a help?

Examples:

  • the return type of a function must have a statically known size
  • unicode escape must not be a surrogate

Joshua Nelson (Sep 05 2021 at 16:56):

I would put those as notes, because they're not suggestions the user can act on

Joshua Nelson (Sep 05 2021 at 16:56):

but it's kind of subjective

Esteban Küber (Sep 08 2021 at 18:01):

Yeah my rule of thumb is note by default when you're informing the user about facts, help when you're letting them know about things they can do
