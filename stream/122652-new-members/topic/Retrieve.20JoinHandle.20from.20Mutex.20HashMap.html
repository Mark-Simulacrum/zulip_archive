First of you need to implement From<ContextValue> instead of From<&ContextValue>. You can't extract an owned JoinHandle from an &ContextValue as JoinHandle can't be cloned. Next to get a ContextValue instead of an &ContextValue you need to use map.remove() instead of map.get(). Finally to allow mutating map you need to use let mut map = ... instead of let map = ....
