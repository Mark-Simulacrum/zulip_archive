The things that seem most interesting to write about are:

  • Sending and sharing data between tasks
  • Synchronizing data across tasks
  • Combinators for non-sequential control flow
  • Cancelling futures by dropping them
  • The poll contract
  • Nested futures and pin projection

I've already written a bunch on "Beware of blocking code" as well

It's probably best if somebody else writes the section on runtime choice, since I'm not exactly neutral on this point.
