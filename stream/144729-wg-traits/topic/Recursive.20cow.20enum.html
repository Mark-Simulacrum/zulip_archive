marmeladema (Apr 23 2021 at 15:02):

Cow definitions is

pub enum Cow<'a, B> where
    B: 'a + ToOwned + ?Sized,  {
    Borrowed(&'a B),
    Owned(<B as ToOwned>::Owned),
}

So its either a reference to [Test<'a>] (which should be Sized) or a Vec<Test<'a>> (which should Sized too)
