Noah Lev (Oct 30 2021 at 00:05):

Also, are there any other sections of the guide that would help those of us new to rustc to prepare for our first gathering?

The "Getting Started" section has information about cloning the repo and building rustc.

Rust Analyzer is very useful since it has go-to-definition, find all references/implementations, and inlay type annotations, though it is definitely possible to read the source code without it. This section of the guide has instructions for setting up Rust Analyzer for use with rustc.

Also, the rustc API docs are quite helpful since you can search for types and functions you see in the source code and quickly get documentation about them. It can also function as a limited version of Rust Analyzer since it has go-to-definition in the source code view (accessed via [src] links).
