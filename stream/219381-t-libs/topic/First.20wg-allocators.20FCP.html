Dear libs team, the wg-allocators has its first FCP: https://github.com/rust-lang/wg-allocators/issues/8

This first change is a "simple" rename of the Alloc trait, and it is hopefully going to help to streamline the process of upstreaming the changes that have consensus within the working group. The idea is that, for unstable functionality, start applying the changes to nightly so that we can gain more practical experience with them - but those changes will obviously go through the FCP process in rust-lang/rust as well, so if you don't have time to chime in the wg-allocators FCPs, don't worry, you'll be pinged when a PR pops up. There is an extensive implementation of all changes with a substantial amount of consensus here: https://github.com/TimDiekmann/alloc-wg

And the wg-allocators repo is full of issues discussing different aspects of the design, so if this topic interests you, feel free to start chiming in!
