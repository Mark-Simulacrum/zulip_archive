The current list of issues in need of a prioritization is here:
https://github.com/rust-lang/rust/issues?q=is%3Aopen+is%3Aissue+label%3AT-compiler+-label%3AP-critical+-label%3AP-high+-label%3AP-medium+-label%3AP-low+label%3AI-prioritize
most of them are regressions found by crater, a diagnostic tool that compiles about ~170K Rust crates found around the internet to see if a release introduces regressions. We run this tool before each stable release. Priority we assign to these issues depends on various factors (which crate breaks, what kind of Rust code does the crate has, etc.)
