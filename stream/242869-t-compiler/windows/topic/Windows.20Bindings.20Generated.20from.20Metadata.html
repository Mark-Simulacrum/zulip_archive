Topic: Windows Bindings Generated from Metadata

rylev (Jan 21 2021 at 17:50):

Announcement: Microsoft has been working on creating bindings to the entire Windows API surface from metadata. It's still early days but this is a huge step to making it easier to maintain the way the entire Rust community interacts with Windows.
Blog: https://kennykerr.ca/2021/01/21/rust-for-windows/
Repo: https://github.com/microsoft/windows-rs
About Metadata: https://blogs.windows.com/windowsdeveloper/2021/01/20/making-win32-apis-more-accessible-to-more-languages/
