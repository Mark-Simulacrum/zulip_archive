You can disable the watch task that is created by rust-analyzer and deinfe one manaully

matklad (Mar 26 2019 at 08:35):

    {
      "label": "cargo watch",
      "group": "build",
      "isBackground": true,
      "type": "shell",
      "command": "cargo",
      "args": [
        "watch"
      ],
      "problemMatcher": "$rustc-watch"
    },
