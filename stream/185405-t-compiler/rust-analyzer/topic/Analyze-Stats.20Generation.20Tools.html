Recently because of my company requirement, I have to study how capable of Github Action.

So I tried to develop a little toy, for a given git commit id of RA , run analysis-stats on 8 differents benchmark projects in rustc-perf (which can config) and generate a report for that (by parsing the text output).
And here is the report site: https://edwin0cheng.github.io/github-action-usage-test/

To add a new commit id, go to https://github.com/edwin0cheng/github-action-usage-test/issues/3 and leave a comment The bot will reply to you when the process is finish.( it use about 15 mins )
