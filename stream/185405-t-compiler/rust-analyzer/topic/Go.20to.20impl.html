Jade (Aug 21 2021 at 09:20):

Aleksey: to answer that, chalk doesn't understand impl functions at all: I'm not even sure if they're in the IR in any way. It does however understand associated types: You can infer Normalize(<SomeType as SomeTrait>::SomeAssocType, ?X) and it will get the value of the associated type from the appropriate impl as the inference variable X. The work I'm doing on chalk is to make a NormalizeFn that will let you ask the same thing but for functions, and it will find you the appropriate function on the impl
