Hey, not sure if people caught it, but it seems that Jetbrains new IDE ("Fleet", closed beta) is using an unspecified LSP server to support Rust: https://www.jetbrains.com/help/fleet/smart-mode.html#workspace-security. I believe this is a departure of their Rust support for CLion.

Yoshua Wuyts [he/they] (Feb 22 2022 at 11:33):

When you enable smart mode, Fleet launches a particular type of backend depending on the language. For example, Java is handled by IntelliJ IDEA, whereas Rust support is provided by a LSP server.
