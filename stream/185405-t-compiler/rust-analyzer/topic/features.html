Sarthak Killedar (Jul 06 2021 at 19:13):

How to get verbose logging from RA?

bjorn3 (Jul 06 2021 at 19:16):

For server side logging set RA_LOG=info. For logging the communication between rust-analyzer and vscode, set "rust-analyzer.trace.server": "verbose in your vscode config.
