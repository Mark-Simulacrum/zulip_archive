Waffle Lapkin (Mar 25 2022 at 15:08):

It seems like https://rust-lang.github.io/rfcs/1946-intra-rustdoc-links.html#linking-to-fields never got to be implemented, right?

What is a proper way of making a link to a field?

Also, using the syntax described in the RFC seems to be broken, for example:

#![warn(rustdoc::broken_intra_doc_links)]

pub struct Struct {
    pub field: (),
}

/// [`field`](Struct.field)
pub fn f() {}

The link links to crate/Struct.field which doesn't exist, no warning produced whatsoever.
