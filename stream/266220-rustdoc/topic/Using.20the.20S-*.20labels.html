Hey all, please remember to switch the S-waiting-on-review/S-waiting-on-author labels when you make/review PRs. I know it's easy to forget, but it makes things easier both for the triage team and for me as a reviewer.

Joshua Nelson (Sep 01 2021 at 00:11):

I think everyone should have permissions to do so from the UI, but if not you can always use @rustbot ready (to set it waiting on review) or @rustbot author (to set it to waiting-on-author): https://github.com/rust-lang/triagebot/pull/1381
