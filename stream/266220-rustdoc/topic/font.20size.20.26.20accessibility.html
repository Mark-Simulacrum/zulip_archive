Right now we specify font size in a variety of ways. In my PR I consolidate on a single way, rem. That's better than em because it solves a problem called "compounding" (if an element of font-size 1.5em is inside an element of font-size 1.5em, the actual size is 2.25 times the document base size). That's better than px because it respects the user's setting for font-size, which some people set larger or smaller for accessibility or preference reasons.

Note that this is distinct from the "Zoom" (Ctrl-+ / Ctrl--) functionality on many (but not all) browsers.

If folks like this I propose to set a rule that font sizes should always be specified in rem, with a small carveout for icon fonts, which should be specified in px.
