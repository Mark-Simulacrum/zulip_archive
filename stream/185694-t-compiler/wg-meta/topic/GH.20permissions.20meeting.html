nikomatsakis (Mar 14 2019 at 16:11):

Roughly speaking I want to be able to

  • easily assign issues to random people who show up and want to take part in a WG
  • as they get more involved, add some people to a list that gives expanded privileges (this may be the same as journeyperson)
  • eventually, add someone to a journeyperson list which gives them r+ and write privileges
  • some time after that, add them to a compiler team that gives them full privileges

I want it to be very simple to apply these same privileges across many repositories.

Similarly, I want to be able to publish new versions of crates to crates.io with ease.

I want maintaining these lists to be something that can be distributed to others, not something only a select few can do.
