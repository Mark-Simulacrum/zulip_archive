I've crated https://github.com/rust-analyzer/minirust to experiment with various flavors of evented parsing.

The idea is to write library-ified parses in different styles for the following language:

File =
  Struct*

Struct =
  'struct' name:'ident' '{' Field* '}'

Field =
  name:'name' ':' ty:'name' ','?
